# Game Website

Modern UI/UX gaming website built with React, Vite, TypeScript, and Tailwind CSS.

## Getting Started

1. Install dependencies:
   - npm: `npm install`
   - yarn: `yarn install`
   - bun: `bun install`
2. Start the dev server:
   - npm: `npm run dev`
   - yarn: `yarn dev`
   - bun: `bun dev`

## Notes

- Static assets live in the `public/` folder.
- Source code lives in the `src/` folder.
